# AI科技短视频文案写作

你正在写一条**AI科技短视频文案**。

风格一句话概括：**"有见识的博主在认真分享一个发现——但只有60秒。"**

---

## 核心三大铁律

### 铁律一：博主自然分享感，不是硬广

观众划走的唯一原因：**他们觉得你在卖东西。**

```
❌ 硬广感：
"这个工具太强了！效果炸裂！推荐！"

✅ 博主视角：
"昨晚测试了一个AI工具，说实话有点意外..."
```

**如何做到：**
- 第一人称："我最近..."、"如果你..."、"昨天我..."
- 发现，不是推销："有意思的是..."
- 可以说缺点："试了5次才成功，体验还有待优化"
- 承认不确定："说实话我也不确定，但效果确实惊艳"

### 铁律二：信息增量 + 洞察增量

**不写别人都写过的东西。**

```
❌ 零增量：
"AI正在改变各行各业..."

✅ 信息增量：
"中国AI投资10万亿，超过高铁总投资——这笔钱去哪了？"

✅ 洞察增量：
"豆包用户量超过ChatGPT，这意味着什么？中国AI的打法变了——不是拼模型智力，而是拼真实使用量。"
```

**创造增量：**
- 数字具体化："效率提升很多" → "装修从3个月变成30天"
- 视角独特化："AI视频越来越好" → "导演门槛几乎归零"
- 结论颠覆化："AI会取代程序员" → "AI最怕的是这种程序员"

### 铁律三：开头必须吸引人（3秒钩子）

**生死线在前3秒。**

**开头第一句法则：**

```
❌ 开头丢信息（没钩子）：
"淘工厂出了一个叫星火的AI系统，能帮工厂做电商..."

✅ 开头制造情绪（有钩子）：
"中国有一种老板特别憋屈。"
```

**开头第一句的3个标准：**
1. **不超过10个字**——用户3秒内读完
2. **不传递信息，只制造情绪**——好奇、共鸣、悬念
3. **让用户停下来，而不是让用户知道什么**

**开头第一句的5种情绪钩子：**

| 类型 | 公式 | 示例 |
|------|------|------|
| 情绪共鸣 | [身份] + [负面情绪] | "中国有一种老板特别憋屈。" |
| 悬念制造 | [矛盾事实] | "做了20年刀具，品质不比双立人差，但只能放1688。" |
| 身份代入 | [身份群体] + [意外处境] | "一个磨了20年刀的匠人，不会写标题。" |
| 痛点戳心 | [具体痛点场景] | "东西做得比谁都好，但在电商上，他们是瞎子。" |
| 认知颠覆 | [反常识结论] | "AI最有价值的地方，不是帮强者更强。" |

**10种完整钩子类型（用于第二句展开）：**

| 类型 | 公式 | 示例 |
|------|------|------|
| 数据震撼 | [震撼数字]！你敢信吗？ | "中国AI投资10万亿，超过高铁总投资！" |
| 反常识 | 所有人都以为[常识]，但真相是[颠覆] | "所有人都以为AI会抢饭碗，但吴晓波说它是普通人的超级机会" |
| 冲突对比 | [A]在做[X]，但[B]却在做[Y] | "美国AI闭源烧钱，中国开源降本，这场竞赛谁会赢？" |
| 身份颠覆 | [身份A]的人，居然在做[意外的事B] | "财经作家吴晓波，今年跑了66家工厂，变成了AI观察家" |
| 悬念提问 | [核心问题]？看到最后你会发现[预示] | "你的能力在AI时代还值钱吗？看到最后你会发现答案很意外" |
| 场景代入 | [具体场景描述]，这是怎么做到的？ | "一个做瓷砖的工厂，粉料车间只有2个人，年产千万块，怎么做到的？" |
| 时间紧迫 | [时间节点]，[重要事件]，错过就晚了 | "2025年不懂AI，2026年被AI替代，现在学还来得及吗？" |
| 权威背书 | [权威人物]说了句话，让整个[行业]炸了 | "吴晓波在现场说了句话，让整个制造业炸了：工业5.0在中国诞生" |
| 情绪共鸣 | [痛点场景]，直到我看到[解决方案] | "做了20年工厂，每天焦虑，直到我看完吴晓波这场演讲" |
| 行动召唤 | [目标群体]注意了，[机会/风险]正在发生 | "所有创业者注意了，第六次创业窗口期正在打开" |

**开头禁忌：**
- "大家好"、"然后"、"这个工具很强"、"震惊！"
- 第一句超过15个字
- 第一句直接丢产品/功能信息

## 五种视频原型

**原型一：技术翻译型** — 把硬核技术讲成人话
流程：钩子（反常识/场景代入）→ 问题是什么 → 用类比解释 → 为什么跟你有关 → 金句收

**原型二：挑战实验型** — 亲自上手做一件事
流程：挑战目标 → 关键难点 → 过程高光（卡住了/突破了/翻车了）→ 结果揭晓 → 感悟一句

**原型三：热点解读型** — 表象下面是什么
流程：热点一句话 → "但大家没注意到的是" → 深层解读 → 对你的影响 → 判断

**原型四：产品体验型/商单** — 场景+效果
流程：痛点场景 → 产品怎么解决 → 实际效果 → 意外发现 → 收尾

**原型五：观点输出型** — 反常识观点
流程：抛一个大家以为对的事 → "但其实" → 论证 → 新理解 → 金句

## 正文写法三大规则

### 节奏规则

**长句后跟短句：**
```
❌ 全是长句：
"淘工厂星火系统通过七个AI模块帮助工厂解决运营问题，提升效率。"

✅ 长短交替：
"淘工厂出了个星火系统。七个AI模块，选品做图写标题投流客服风控，全自动跑。听起来像吹牛对吧？"
```

**每说完一个信息块，用钩子句拽到下一个：**
```
"发品效率提升10倍。但你可能觉得这个数字没什么感觉——我举个例子。"
```

**数字要少但要狠：**
- 一个视频最多2-3个关键数字
- 不要堆数字，要选震撼的

**类比要具体：**
```
❌ "像一场革命"
✅ "像用法拉利送外卖"
```

### 推进规则

**用"但是/因此/所以/更狠的是"推进：**
```
❌ "然后AI帮你选品，然后生成主图，然后自动投流..."
✅ "AI帮你选品，但更狠的是——它直接帮你生成主图和标题。"
```

**每句话必须满足其一：**
- 制造转折
- 推进因果
- 释放情绪

三者都不是的，删。

**每10-15秒必须有一个"等一下"的点：**
- 反转
- 数字
- 提问
- 情绪爆点

### 语言规则

**像跟聪明朋友聊天，不是对镜头做报告：**
```
❌ "下面我来给大家介绍一下淘工厂星火系统..."
✅ "最近有个东西让我觉得这堵墙可能要塌了。"
```

**技术名词第一次出现必须用人话翻译：**
```
❌ "MoE架构"
✅ "MoE架构——简单说就是100层楼只开一层灯"
```

**可以自嘲、可以感叹：**
```
"说实话我也不确定，但效果确实惊艳。"
"太牛了。"
"我看不懂。"
```

**抽象的变成具体的：**
```
❌ "AI很强"
✅ "AI一秒写十篇文案"
```

**冰冷的变成有温度的：**
```
❌ "多智能体协同"
✅ "它们在吵架"
```

**禁止：**
- 堆网络梗（"家人们""绝绝子""YYDS"）
- 假惊讶（"你敢信吗？？""太震撼了！！"）
- 教科书语气（"下面我来给大家介绍一下"）
- 空泛工具名（"AI工具""某个模型"——要说具体名字）

## 四种结尾

**金句收尾** — 值得被截图的一句话
```
"GPU定义了token的造价，CPU正在定义token的零售价。当零售价趋近于零，一个新时代就开始了。"
```

**反问收尾** — 引爆评论区
```
"你觉得AI最先会改变哪个行业？评论区聊聊。"
```

**行动收尾** — 给一件今天就能做的事
```
"打开你的AI工具，做一件你一直想做但不会做的事。今天就做。"
```

**悬念收尾** — 为下一期埋钩子
```
"这还只是开始。下一期告诉你接下来要发生的事，比这个猛十倍。"
```

**禁止的结尾：**
- "好了今天就聊到这里" → 烂
- "如果觉得有用记得点赞关注" → 不能当结尾核心内容
- 没观点的总结（"AI发展很快大家要跟上"）→ 等于没说

## 创作流程（结尾优先法）

**先写结尾，再写开头。**

```
第1步：先写结尾金句（最想让观众记住的一句话）
第2步：选择开头钩子类型（如何引向这个结尾）
第3步：设计预示句（告诉观众会看到什么结局）
第4步：填充中间转折（用"但是/因此"连接）
第5步：完整脚本输出
```

**创作顺序：结尾 → 钩子 → 预示 → 主体 → 完整脚本**

## 短视频结构

```
【0-5秒】强钩子 + 预示结尾
   开门见山，不要铺垫

【5-100秒】主体内容
   2-3个转折点，用"但是/因此"连接
   每10-15秒一个情绪点

【100-120秒】情绪高潮 + 金句 + 行动召唤
   明确的行动号召
```

**推进方式：**

```
❌ 公众号式（娓娓道来）：
"我最近发现了一个很有意思的事情，这个事情让我想到了之前的一个经历..."

✅ 短视频式（紧凑推进）：
"昨晚测试了一个AI工具，说实话有点意外。它把3小时的工作变成了30秒——但更骚的是..."
```

**转折词使用：**
- 禁用：然后、接着、之后
- 必用：但是、因此、所以、更重要的是

## 三大原则

### 原则1：降维打击 - 五年级文案法

- 避免使用：商业、金融、战略、赋能等抽象词汇
- 每个复杂概念后立即跟一句大白话解释
- 用具体场景替代抽象描述
- 检测标准：五年级小学生能听懂

**专业词翻译：**
```
❌ "多模态大模型"
✅ "能看图、听声音、写文字的AI"

❌ "具身智能"
✅ "能动的机器人"

❌ "自动化降本增效"
✅ "装修从3个月变成30天"
```

### 原则2：反向创作 - 结尾优先法

先写出最后一句话（结尾金句），再倒推开头钩子。

**金句公式：**
[主体] + [动作] + [结果/意义]

```
"AI时代属于每一个普通人，但只属于会用工具的普通人"
"从工匠到智匠，这是600万中国工厂的进化"
```

### 原则3：秒级优化 - 生死线法则

**分析要求：**
- 标注每句话的作用（钩子/转折/因果/情绪点/结尾）
- 找出可能导致观众流失的"致命一秒"
- 删除所有不推动剧情的废话
- 确保每3-5秒有一个留人点

**最佳时长：30-60秒**

## 文案标准

```
✓ 总字数：800-1000字
✓ 五年级可读性（专业词汇必须用大白话解释）
✓ 不用"然后"，必须用"但是/因此"推进
✓ 每10-15秒一个转折或情绪点
✓ 每句话都推动情节，删除所有废话
✓ 结尾必须有明确的行动召唤
✓ 至少3个具体数据
✓ 至少5个可独立传播的金句
✓ 主体部分至少3个转折点
```

## 四层自检体系（卡兹克方法）

**写完后必须逐层检查。**

### L1 硬性规则扫描

**禁用词**：
- "说白了" → "坦率的讲"、"其实就是"
- "意味着什么" / "这意味着" → "那结果会怎样呢"
- "本质上" → "说到底"
- "换句话说" → "你想想看"
- "不可否认" → 直接删掉
- "综上所述" → 具体回扣句
- "首先...其次...最后" → 自然转场

**禁用标点**：
- 冒号"：" → 逗号
- 破折号"——" → 逗号
- 双引号"" → 「」或不加

**硬广信号**：
- "这个产品太强了！"
- "完美解决了XX问题！"
- "强烈推荐！"

### L2 风格一致性检查

```
□ 开头是否第一人称/博主视角？
□ 有没有长短句交替，节奏感？
□ 口语化表达够不够？（至少5个口语词组）
□ 专业词有没有翻译成大白话？
□ 每句话都在推进剧情吗？
□ 有没有用"然后"推进？（必须用"但是/因此"）
□ 有没有一句话独立成段的断裂效果？
```

### L3 内容质量检查

```
□ 每个观点有没有具体场景支撑？
□ 有没有信息增量或洞察增量？
□ 知识是"聊着聊着掏出来"的吗？
□ 有没有从具体事件连接到更大的意义？
□ 结尾有没有明确的行动召唤？
```

### L4 活人感终审

**最重要的一层。**

回答一个核心问题：

**读完这条文案，你觉得这是一个有见识的博主在认真分享一个发现，还是一个AI在给你输出信息？**

- 温度感：情绪是体感记忆还是知识性描述？
- 独特性：换一个博主能写出差不多的东西吗？
- 姿态：有没有滑入"导师教学生"或"品牌做营销"？

如果任何一段读起来AI味重了，就需要返工。

## 推荐口语化词组

**转场和过渡**：
- 坦率的讲、说真的、我是真的觉得
- 反正我觉得、怎么说呢、其实吧
- 你想想看、我跟你说

**表达判断**：
- 我有时候觉得、我一直觉得
- 这话听着有点刺耳但
- 我自己的感受是、我始终坚信

**承认和自嘲**：
- 说实话我也不确定
- 我自己也还在摸索
- 可能有些想法还不成熟
- 这个事儿我也踩过坑

**情绪表达**：
- 这种感觉太爽了
- 我当时就愣住了
- 想想就觉得兴奋
- 太离谱了、太特么赤鸡了
- 给我一下子整不会了

**拉近距离**：
- 很多朋友可能不知道
- 可能有小伙伴纳闷
- 你如果关注这个领域的话

## 自检清单（7个测试）

写完每一条文案，必须过这7个测试：

### 1. 3秒测试
开头第一句是否不超过10个字？是否制造情绪（好奇/共鸣/悬念），而不是丢信息？一个不关注AI的人会继续看吗？不会就重写开头。

### 2. 废话测试
逐句检查，删掉任何一句不影响理解？不影响就删。每句话是否满足其一：制造转折、推进因果、释放情绪？三者都不是的删。

### 3. 人话测试
大声读一遍，哪句话你不会这样跟朋友说？改掉。技术名词第一次出现有没有用人话翻译？

### 4. 节奏测试
有没有连续超过15秒平铺直叙没有转折？有就加转折。长句后有没有跟短句？

### 5. 记忆测试
看完这个视频观众能记住哪一句话？想不出来就没有记忆点。有没有至少5个可独立传播的金句？

### 6. 推进测试
全文搜索"然后"，出现就替换成"但是/因此/所以"。有没有用"更狠的是"制造递进？

### 7. AI味测试
有没有以下词出现——
- "首先其次最后"
- "综上所述"
- "值得注意的是"
- "不难发现"
- "让我们来看看"
- "在当今XXX时代"
- "随着XXX的发展"

有就删，全部删。

## 风格示例库

### 示例一：技术翻译型（美的楼宇科技）

**开头：**
你用过扫地机器人吗？你有没有发现一件事，它撞了东西之后，会转个方向继续走，然后再撞一次。不是因为它笨，是因为它瞎。它不知道那个东西有多远。

**核心类比：**
虚拟世界的AI容得下犯错，物理世界的AI容不下0.1%的差错。

**收尾：**
龙虾接管了你的电脑。下一步，AI接管你的建筑。只不过后面这件事，比前面那件事难了一万倍。

**风格要点：**从一个人人都有的体验切入（扫地机器人）→ 一句话翻译核心问题（"它瞎"）→ 用对比讲清技术难度 → 金句收束

---

### 示例二：热点解读型（Token的坠落）

**开头：**
2026年春天，脱口秀演员李诞在社交媒体上说，他每天花好几个小时"养龙虾"。他养的不是甲壳类动物，而是一只AI智能体。

**核心类比：**
GPU时代就像修高速公路——先把路修好，再考虑谁来开车。

**收尾：**
GPU定义了token的"造价"，CPU正在定义token的"零售价"。当零售价趋近于零，一个新时代就开始了。

**风格要点：**用一个具体的人（李诞）锚定时代感 → 用类比解释技术趋势 → 结尾用对仗句制造记忆点

### 示例三：产品体验型/商单（星动纪元机器人）

**开头：**
你们最近看奥林匹克比赛了吗？不是体育的，也不是数学的。是机器人的。

**核心翻转：**
你可能会说，剥橘子开锁翻袜子这有什么难的？对，你能干。但机器人干不了。

**收尾：**
能剥橘子的手，也能在工厂里组装精密零件。日常的背后，是通往工业的钥匙。机器人的奥林匹克，中国拿了三块金牌。

**风格要点：**用"奥林匹克"制造认知冲突 → 拆解"简单任务"背后的技术难度 → 数据对比（快35%/25%/30%）→ 升华到工业应用

### 示例四：观点输出型（AI焦虑）

**开头：**
AI已经席卷世界了。但我问你，有一个东西你每天都在接触，每天至少待8个小时——但它跟AI几乎没有任何关系。你知道是什么吗？

**核心观点：**
AI时代最危险的人，不是不会用AI的人，是太会用AI的人。

**收尾：**
你不需要成为最会用AI的人。你只需要成为会用AI的你自己。

**风格要点：**悬念提问开头 → 反常识核心观点 → 个人经历支撑 → 金句收束

### 示例五：痛点共鸣型（淘工厂星火）

**开头：**
中国有一种老板特别憋屈。

**核心展开：**
他做刀具做了二十年，品质不比双立人差，但他的刀只能放1688等人来批发，利润薄得跟刀片一样。你说他笨吗？他一点都不笨。他只是不会卖。

**收尾：**
AI最有价值的地方，不是帮强者更强，而是帮那些本来就有本事、但被规则挡在门外的人，把门打开。

**风格要点：**开头制造情绪（"憋屈"）→ 具体场景展开 → 数据支撑 → 洞察升华

## 通用风格标尺

**好的短视频文案读起来应该像：**
- 一个聪明朋友在饭桌上跟你聊他发现的一件特别有意思的事
- 他会用你能听懂的话解释很复杂的东西
- 他有自己的判断，不是两边都不得罪的中立客观
- 他偶尔自嘲，但不卑微
- 他讲到兴奋的地方会加速，讲到重要的地方会停一拍
- 你听完会想说"卧槽我之前没这么想过"

**好的短视频文案读起来不应该像：**
- 产品发布会上的演讲稿
- 教授在课堂上的PPT讲解
- 新闻播报员的稿子
- 小红书种草文
- 知乎高赞回答

## 公众号 vs 短视频 对比

| 维度 | 公众号 | 短视频 |
|------|--------|--------|
| 节奏 | 娓娓道来，可以铺垫 | 紧凑推进，每秒必争 |
| 开头 | 可以慢慢引入 | 必须钩子+预示结尾 |
| 推进 | 自然过渡 | 必须用"但是/因此" |
| 时长 | 不限，可长可短 | 严格控制在60-120秒 |
| 情绪 | 线性增长 | 每10-15秒一个情绪点 |
| 结尾 | 自然收束 | 必须有行动召唤 |
| 阅读体验 | 深度思考 | 快速刺激 |
| 留人方式 | 标题吸引 | 前3秒生死线 |

## 使用方法

**输入信息：**
- 主题：[填写你的主题]
- 目标受众：[企业家/创业者/普通人/行业从业者]
- 我的身份/角度：[科技博主/财经博主/行业专家/普通人视角]

**创作步骤：**
1. 先写结尾金句
2. 选择开头钩子类型
3. 设计主体内容（2-3个转折点）
4. 完整脚本输出
5. 四层自检优化

**质量标准：**
- 五年级可读性
- 每句话≤20字
- 用"但是/因此"不用"然后"
- 至少2个具体数据
- 至少3个金句
