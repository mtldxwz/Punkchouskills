# AI科技短视频文案写作 Skill

专为AI科技领域打造的短视频文案写作技能，针对短视频快节奏特点设计。

## 核心特点

### 三大铁律

1. **博主自然分享感，不是硬广**
   - 第一人称："我最近..."、"如果你..."、"昨天我..."
   - 发现，不是推销："有意思的是..."
   - 可以说缺点、承认不确定

2. **信息增量 + 洞察增量**
   - 不写别人都写过的东西
   - 数字具体化、视角独特化、结论颠覆化

3. **开头必须吸引人（3秒钩子）**
   - 开头第一句不超过10个字
   - 不传递信息，只制造情绪
   - 让用户停下来，而不是让用户知道什么

### 五种视频原型

- **技术翻译型**：把硬核技术讲成人话
- **挑战实验型**：亲自上手做一件事
- **热点解读型**：表象下面是什么
- **产品体验型/商单**：场景+效果
- **观点输出型**：反常识观点

### 正文写法三大规则

1. **节奏规则**：长句后跟短句、数字要少但要狠、类比要具体
2. **推进规则**：用"但是/因此/所以"推进，禁用"然后"
3. **语言规则**：技术名词翻译、抽象变具体、冰冷变有温度

### 四种结尾

- **金句收尾**：值得被截图的一句话
- **反问收尾**：引爆评论区
- **行动收尾**：给一件今天就能做的事
- **悬念收尾**：为下一期埋钩子

### 7个自检测试

1. 3秒测试：开头是否制造情绪？
2. 废话测试：每句话是否推动剧情？
3. 人话测试：你会这样跟朋友说吗？
4. 节奏测试：有没有连续15秒平铺直叙？
5. 记忆测试：观众能记住哪一句话？
6. 推进测试：有没有用"但是/因此"？
7. AI味测试：有没有教科书语气？

## 使用方法

1. 加载skill：将SKILL.md内容作为写作指导
2. 创作流程：
   - 第1步：找到你的增量
   - 第2步：先写结尾金句
   - 第3步：设计开头信息缺口
   - 第4步：写主体（2-3个转折点）
   - 第5步：念出来自检

## 适用场景

- 抖音/B站1-3分钟短视频
- AI科技领域内容
- 商单/合作视频文案

## 字数标准

- 30-60秒视频：约200-350字
- 1-2分钟视频：约350-600字
- 3-5分钟视频：约800-1500字
